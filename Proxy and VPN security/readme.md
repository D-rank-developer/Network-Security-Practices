# Proxy and VPN Security

## Prerequisites
You will need the following virtual machines fully set up and running:

- **Gateway**
- **Workstation**
- **Webserver**
- **Zabbix**

---

## Lab Tasks

- **Install and configure Squid proxy server on the Gateway**
- **Implement access controls and content filtering**
- **Analyse proxy logs to understand user behavior**
- **Install and configure OpenVPN server on the Gateway**
- **Generate and manage certificates for VPN authentication**
- **Configure the VPN client on the Workstation and establish an encrypted tunnel**
- **Compare network traffic with and without VPN using packet analysis**
- **Understand the security implications and ethical considerations**
